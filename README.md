# study-app-kanji

小学1年生向けの漢字熟語ゲーム

## 概要

80個の小学1年生で習う漢字カードから2枚を選んで熟語を作るゲームです。
制限時間3分以内にできるだけ多くの熟語を作って高得点を目指しましょう。

## 遊び方

1. 手札16枚から2枚の漢字カードを選択
2. 正しい熟語になれば得点獲得、新しいカードを補充
3. 3分間でできるだけ多くの熟語を作る

### 得点

| 種類 | 得点 |
|------|------|
| 一般熟語 | 2点 |
| 苗字・数字 | 1点 |
| 逆順ボーナス | +2点 |

### ボタン

- **すてる**: 選択したカード1枚を捨てる
- **ぜんぶすてる**: 手札を全て入れ替える
- **おわる**: ゲームを終了する

## 開発

```bash
npm install      # 依存関係インストール
npm run dev      # 開発サーバー起動
npm run build    # プロダクションビルド
npm run preview  # ビルド結果プレビュー
```

## 技術スタック

- React 19
- TypeScript 5
- Vite 7
- CSS Modules

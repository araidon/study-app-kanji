# 漢字熟語ゲーム

小学1年生向けの漢字熟語ゲーム

## 概要

80個の小学1年生で習う漢字カードから2枚を選んで熟語を作るゲームです。
制限時間内にできるだけ多くの熟語を作って高得点を目指しましょう。

## 遊び方

1. 手札16枚から2枚の漢字カードを選択
2. 正しい熟語になれば得点獲得、新しいカードを補充
3. 制限時間内にできるだけ多くの熟語を作る

## 得点

| 種類 | 得点 |
|------|------|
| 一般熟語 | 2点 |
| 数字 | 1点 |

## 設定

| 項目 | 範囲 | デフォルト |
|------|------|-----------|
| 制限時間 | 1〜10分 | 5分 |
| ヒント | ON/OFF | ON |
| ヒント表示 | 5〜60秒 | 20秒後 |

ヒントをONにすると、一定時間操作がない場合に正解となるカードが光ります。

## ボタン

- **すてる**: 選択したカード1枚を捨てる
- **ぜんぶすてる**: 手札を全て入れ替える
- **おわる**: ゲームを終了する

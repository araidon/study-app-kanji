# 漢字熟語ゲーム 仕様書

## 概要

小学1年生で学ぶ漢字80字を使って2字熟語を作るブラウザゲーム。
カードゲーム形式で、制限時間内にできるだけ多くの熟語を作ることを目指す。

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | React 19 + TypeScript |
| ビルドツール | Vite 7 |
| スタイリング | CSS Modules |
| ホスティング | Cloudflare Pages |

## 画面構成

### 1. タイトル画面
- ゲームタイトル「漢字熟語ゲーム」
- サブタイトル「小学1年生のかんじ80字」
- 「ゲームスタート」ボタン
- 遊び方の説明

### 2. ゲーム画面
- **情報バー**: 残り時間、得点、山札枚数
- **結果バナー**: 正解/不正解のフィードバック表示
- **手札エリア**: 2行5列（10枚）のカード配置
- **ゴミ箱**: 不要なカードを捨てるボタン

### 3. 結果画面
- 最終得点
- 作成した熟語数
- 作成した熟語一覧（得点内訳付き）
- 「もういちど あそぶ」ボタン

## ゲームルール

### 基本ルール
1. ゲーム開始時に80枚のカードをシャッフルし、10枚を手札として配布
2. カードを2枚選択して熟語を作成
3. 1枚目→2枚目の選択順で熟語を判定
4. 正解なら得点加算、2枚消費、2枚補充
5. 不正解なら選択解除（ペナルティなし）
6. 不要なカードはゴミ箱に捨てて1枚補充

### 終了条件
- 制限時間（3分）経過
- 山札が0枚になり、手札が10枚未満

### 得点システム

| 熟語タイプ | 点数 | 例 |
|------------|------|-----|
| 一般的な熟語 | 2点 | 火山、天気 |
| 苗字 | 1点 | 山田、田中、赤木、三木 |
| 数字関連 | 1点 | 十円、十字、三十、五十 |
| 逆順ボーナス | +2点 | 左右（逆順で「右左」も成立） |

## データ仕様

### 漢字データ（80字）
```
一右雨円王音下火花貝学気九休玉金空月犬見五口校左三山子四糸字耳七車手十出女小上森人水正生青夕石赤千川先早草足村大男竹中虫町天田土二日入年白八百文木本名目立力林六
```

### 熟語データ構造
```typescript
interface Jukugo {
  word: string      // 2文字の熟語
  type: 'surname' | 'general'  // 苗字 or 一般
  reverse?: string  // 逆順で成立する熟語（ボーナス対象）
}
```

## UI/UX仕様

### タッチ操作
- カードタップで選択/選択解除
- 選択中のカードは視覚的にハイライト
- ゴミ箱タップで選択中のカードを捨てる

### iPhone Safari対応
- セーフエリア対応（ノッチ対策）
- 100dvh使用（アドレスバー問題対策）
- オーバースクロール防止
- ダブルタップズーム無効

### 視覚的フィードバック
- 選択中: 青枠 + 拡大
- 正解時: 緑バナー表示
- 不正解時: 赤バナー表示
